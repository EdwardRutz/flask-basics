# Flask Basics

> Demo Flask basics

- Creating and controlling views and routes in Flask
- Setting and reading from cookies
- Rendering templates
- Template inheritance

## Dependencies

- Python
- Flask


## Setup

- Install Flask, `pip install Flask`
- In REPL, check flask version
```python
import flask
flask.__version__
```
